# PDF Processing Platform Features

This document outlines the 25 features implemented in the PDF Processing Platform.

## Core Manipulation
1.  **Merge PDF**: Combine multiple PDF documents into a single file.
2.  **Split PDF**: Split a PDF document into individual pages or specific page ranges.
3.  **Compress PDF**: Reduce the file size of a PDF document while maintaining acceptable quality.
4.  **Rotate PDF**: Rotate specific pages or the entire document (90, 180, 270 degrees).
5.  **Organize PDF**: Reorder pages within a PDF or delete specific pages.
6.  **Repair PDF**: Attempt to repair corrupted or damaged PDF files using Ghostscript.

## Conversion (To PDF)
7.  **Word to PDF**: Convert Microsoft Word documents (.docx) to PDF.
8.  **PowerPoint to PDF**: Convert Microsoft PowerPoint presentations (.pptx) to PDF.
9.  **Excel to PDF**: Convert Microsoft Excel spreadsheets (.xlsx) to PDF.
10. **JPG to PDF**: Convert image files (JPG, PNG) to a single PDF document.
11. **HTML to PDF**: Convert HTML files or content to PDF.
12. **Scan to PDF**: Convert scanned images to PDF with optional image enhancement (contrast improvement).

## Conversion (From PDF)
13. **PDF to Word**: Convert PDF documents to editable Microsoft Word files (.docx).
14. **PDF to PowerPoint**: Convert PDF slides to editable PowerPoint presentations (.pptx).
15. **PDF to Excel**: Extract tables from PDF documents into Excel spreadsheets (.xlsx).
16. **PDF to JPG**: Convert PDF pages into high-quality JPG images.
17. **PDF to PDF/A**: Convert standard PDFs to PDF/A format for long-term archiving.

## Editing & Security
18. **Edit PDF**: Add text annotations, shapes, and drawings to PDF pages.
19. **Sign PDF**: Add digital signatures (image-based) to PDF documents.
20. **Watermark PDF**: Add text watermarks to PDF pages with customizable opacity and rotation.
21. **Unlock PDF**: Remove password protection from encrypted PDFs (requires correct password).
22. **Protect PDF**: Encrypt PDF documents with a password.
23. **Page Numbers**: Add page numbers to PDF documents with customizable positioning.

## Advanced Features
24. **PDF OCR**: Extract text from scanned or image-based PDFs using Tesseract OCR.
25. **Compare PDF**: Compare two PDF documents and generate a visual difference report.
